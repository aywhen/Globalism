# 全球化主义－核心篇

## １、前言

```
全球化主义，以现在的角度来说，属于超现代主义，为什么如此说？
因为目前世界格局要实现真正意义上的全球化主义，还尚早。
为了让世界变成完全统一的国度，人类文明路还很长。
还有许多问题要解决：

1. 消灭爱国主义
2. 消灭各种歧视
    * 消灭种族主义
    * 消灭至上主义
3.消灭战争
```

# ３、正文

一、所有术语的解释不能违背核心内容的解释，但可以进行具体化解释和扩充化解解释。

二、全球化主义的至上准则是，平等

三、平等的解释
* 每个人都享有同等的机会
* 每个人都享有同等的权利
* 每个人都享有同等的地位

四、政治的解释
* 政，众人之事；在国家范畴内指，全体人民利益相关的事务；简称，公务。
* 政治，公务的处理。

五、参与政治，必须坚持人人平等的原则。	

六、规则的解释
* 法律，保证性规则。如果违反法律，将接受人民的审判；如果抵抗执法，人民可采取强制伏法的手段。
* 宪法，属于法律，但是其他法律的制定，必须不能违背宪法。
* 合同、协议，签署保障性规则。内容的起草必须首先遵循法律，必须在自我意愿的情况下签署，签署后将成为保障性规则；失效后，将失去保障性。

七、立法的解释
* 属于政治
* 商讨、制定、表决和颁布法律。
* 对法律进行最终的解释。

八、法律的精神：维护平等

九、权力的解释
* 表决权，对于政治处理方式、方法的决断权。
* 每个人的表决，体现了每个人意识的重要性；这是一种意识力量；简称，意志。
* 权力，是全体人民的意志。

十、代表制的解释
* 选举代表属于政治
* 为了提高执政的合理性、专业性、高效性，由人民选举代表，并授予代表合法的权力进行执政。
* 代表，执政长官。譬如总统、法官，检察官等等。

十一、弹劾的解释
* 属于政治
* 如果代表的执政行为不符合人民利益或者违反了法律，人民有权弹劾代表。

十二、民主的解释
* 人人拥有参政的机会，人人拥有参政的权利。
* 代表的执政权力由人民授权，具备合法性。

十三、自由的解释
* 首先必须遵守法律
* 自由言论、结社、游行、罢工等等都可以通过制定在法律中
* 制定一套优秀的法律，才能体现人民期望的自由。

十四、重释，对于全球化主义核心内容，如果违背了人民大众的意愿，人民有权进行重新解释。
